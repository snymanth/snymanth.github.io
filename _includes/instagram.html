Big Data Visualisation

A group project completed as part of a M.Eng degree at the University of Witwatersrand. A web based dashboard was developed to visualise a publicly available "big data" data source from the World Bank.

The dashboard was developed using Node.js technology and makes use of the Express.js framework.

The author was allocated the responsibility of implementing the framework and structure that the web application would use. In addition to this the author was also tasked to develop the indicator dashboard and maintain the project wiki (located here). A comprehensive help page providing the members with an overview of the framework and how to implement their dashboards using the framework was developed (located here).

Road Network Maintenance

A group project completed as part of a M.Eng degree at the University of Witwatersrand. This Java application models the domain of a system to aid the transport department of local government in maintaining their assets, which include not only the road network but also supporting infrastructure.

The application was developed using Behavior Driven Development (BDD) and Test Driven Development (TDD) approaches. Cucumber and JUnit were used as testing frameworks. The design methodology used was Domain Driven Design (DDD).
